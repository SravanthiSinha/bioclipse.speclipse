Getting started with Speclipse

Introduction

Speclipse is an extension for Bioclipse to work with spectral data. Since you read this, you have installed it properly. Here, we show you what you can now do

Install sample data

You should install the sample data. This can be done by choosing File->New ->Data->Speclipse Sample Data Project from the menu. Leave the defaults and choose Next and Finish. The new Project will contain two subfolders, which represent the two data types Speclipse handles: spectra and assigned peak spectra.

Work with spectra

This screenshot shows several elements of what you can do with spectra:

  • Continuous spectrum and peak visible, peaks can be picked and the integral shown
  • Metadata are visible in the outline and can be edited on other tabs
  • You can save in different formats

Go to Help->Cheat Sheets->Spectra->General Spectrum Cheat Sheet to get more details on this

Assigend Spectra

In Speclipse, you can combine a structure and one or more spectrum in one CML file and assign the peaks. The screenshot shows a file opened in the editor and some hints on what you can do. See the Assigned Spectrum help for details.

Scripting

The functionalities of the plugins are also available via the Bioclipse scripting language. See the general scripting help and the Speclipse scripting help for details. The following script is an example, which loads a continuous spectrum and picks its peaks and saves the result as a CML file:

contspectrum = spectrum.loadSpectrum("/Sample Data1/Speclipse/spectra/aug07.dx")
peaks = spectrum.pickPeaks(contspectrum)
spectrum.saveSpectrum(peaks, "/Sample Data 1/test.cml", "cml")
