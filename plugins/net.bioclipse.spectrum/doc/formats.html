File formats supported by the spectrum plug-in

The spectrum plug-in supports reading and writing of the following file formats:

  • CMLSpec - The spectrum encoding part of CML, the Chemical Markup Language
  • JCamp-DX - The IUPAC Standard Exchange Format for Spectroscopic Data
Within the spectrum plug-in the Java implementation of CML is used for the internal representation of spectroscopic data and for the import and export of structures and spectra to and from the CML file format. Parsing and generation of jcamp-dx files is realized by using the jcampdx-lib (the reference implemention of the IUPAC JCAMP-DX spectroscopy data standard). This library can be found at http://sourceforge.net/projects/jcamp-dx.
