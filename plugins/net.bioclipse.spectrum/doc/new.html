The "New Spectrum" Wizard

If you want to create a new spectrum, you have to select from the new dialog, which you reach by selecting "File - New..." or pressing Ctrl+N. Now the new Spectrum Wizards starts pops up its first dialog.
With the help of the Wizard you are now able to create a new spectrum file in any folder of your current projects in one of the supported formats. The second wizard page gives you the possibility to set the spectrum type to one of the following types:

  • Mass spectrum
  • NMR spextrum
  • IR spectrum
  • UV specctrum
and to add peaks to the newly created spectrum if you like. You can also upload a CSV (comma separated values)-style list of peaks. The format must be (separator)*XValue(separator)+YValue(separator)*newline.
